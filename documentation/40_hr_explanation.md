# 40. Project Explanation for HR

## 👶 2-Minute Explanation
"I built an AI-Powered Job Application Tracker. It helps job seekers keep track of their job applications, companies, and recruiter contact details. It has a dashboard that displays statistics and a Kanban board to track application progress. It also integrates with AI (Gemini and OpenAI) to analyze resumes against job descriptions, write outreach emails, and generate custom interview questions based on the job role."

---

## 🧑 5-Minute Explanation
"This project is a full-stack web application designed to optimize the job search workflow. The backend is built using Java, Spring Boot, and Hibernate, storing data in a MySQL database. The frontend is built using React and Vite, using a responsive Glassmorphism layout. It includes relational tracking for companies and recruiter contacts. It has an Excel export feature using Apache POI, and an AI Suite that makes REST calls to OpenAI and Google Gemini APIs to score resumes, write emails, and generate interview guides."
