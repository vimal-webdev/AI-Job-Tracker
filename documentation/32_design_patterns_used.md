# 32. Design Patterns Used

## 👶 For Freshers (Simple English)
Design patterns are standard templates used to solve common coding problems. They make code easy to read and scale.

---

## 🧑•💻 For Intermediate Developers (Technical Explanation)
- **Dependency Injection**: Spring manages bean instantiation and injection automatically.
- **Factory Pattern**: Dynamically instantiates the correct `AIService` implementation depending on active property files.

---

## 👴 For Senior Developers & Interview Prep (System Design & Code Quality)
- **Builder Pattern**: Created via Lombok's `@Builder` annotation to construct complex entity and DTO objects.
- **Template Method Pattern**: Used by Spring's `RestTemplate` to handle generic REST operations, abstracting connection management and serialization.\n
