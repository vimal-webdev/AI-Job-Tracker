# 31. Performance Optimization

## 👶 For Freshers (Simple English)
Optimization is making the code run faster and use less memory. For example: loading database results only when needed, or compiling frontend files to be small.

---

## 🧑•💻 For Intermediate Developers (Technical Explanation)
- **Lazy Loading**: Hibernate lazy loading prevents fetching unnecessary joined records.
- **React Rendering**: Avoid inline function definitions inside components to prevent unnecessary re-renders.

---

## 👴 For Senior Developers & Interview Prep (System Design & Code Quality)
- **Database Connection Pooling**: Fine-tune HikariCP settings (connection timeout, max lifetime) to optimize connection handling.
- **Query Optimization**: Create composite indexes on foreign keys to speed up search filter query executions.\n
