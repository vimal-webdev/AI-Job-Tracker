# 34. OOP Concepts Used

## 👶 For Freshers (Simple English)
Object-Oriented Programming (OOP) uses objects to represent data and logic. Key concepts: Inheritance, Polymorphism, Encapsulation, Abstraction.

---

## 🧑•💻 For Intermediate Developers (Technical Explanation)
- **Encapsulation**: Using private variables with public getter/setter methods.
- **Abstraction**: Hiding complex database/AI code behind simple service interfaces.

---

## 👴 For Senior Developers & Interview Prep (System Design & Code Quality)
- **Polymorphism**: Used to swap between AI providers at runtime.
- **Inheritance**: Base entity templates can be extended to share common columns (like timestamps or IDs).\n
